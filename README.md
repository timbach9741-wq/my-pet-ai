# 반려동물 AI 건강 진단소 (Pet AI Health Check)

## 프로젝트 개요
이 프로젝트는 **Teachable Machine**과 **TensorFlow.js**를 활용하여 반려동물(강아지)의 사진을 분석하고, 비만 여부를 진단하는 웹 애플리케이션입니다.

## 사업 분석 (Business Analysis)

### 1. 목표 시장 및 타겟 (Target Audience)
*   **주요 사용자**: 반려견의 체중 관리와 건강 상태에 관심이 많은 20-40대 반려인.
*   **니즈 (Needs)**: 병원을 방문하지 않고도 간편하게 반려동물의 건강 상태를 확인하고 싶어함.

### 2. 가치 제안 (Value Proposition)
*   **간편함**: 사진 한 장으로 즉시 비만 여부 확인.
*   **접근성**: 웹 기반으로 별도의 앱 설치 없이 사용 가능.
*   **맞춤형 조언**: 진단 결과에 따라 적절한 관리법(식이요법, 운동 등) 제공.

### 3. 수익 모델 (Monetization Strategy)
*   **제휴 마케팅 (Affiliate Marketing)**: 진단 결과에 따라 맞춤형 사료나 영양제 추천 링크(예: 쿠팡 파트너스)를 제공하여 수수료 수익 창출.
    *   **비만 진단 시**: 다이어트 사료, 저칼로리 간식 추천.
    *   **정상 진단 시**: 영양제, 건강 유지용품 추천.
*   **프리미엄 기능 (추후 확장)**: 수의사 상담 연결, 상세 건강 리포트 제공.

## 기술 스택
*   HTML5 / CSS3
*   JavaScript (ES6+)
*   TensorFlow.js (@teachablemachine/image)

## 설치 및 실행
1. 이 저장소를 클론합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다.
3. 사진을 업로드하여 분석을 시작합니다.
